{[%hh_hdr_ru:Мастер добавления изображений :: Протокол добавления]}

Здесь отображаются последовательно все операции, произведённые с файлами изображений. Каждый файл из числа выбранных для добавления либо помечается как успешно добавленный, либо его запись содержит текст произошедшей ошибки. Кроме того, в первом случае присутствует информация о результатах автозаполнения свойств изображения. {[%hh_h2:Элементы интерфейса]}
Отображать только ошибки Если этот переключатель включен, список протокола отображает только сбойные файлы, в противном случае показываются также успешно добавленные изображения. Включить фильтрацию полезно в случае, если добавлялось множество файлов, из которых только немногие дали сбои (надеюсь, так в большинстве случаев и будет).
Сохранить в файл... Позволяет записать протокол в простой текстовый файл, полезный, например, для последующего просмотра или машинной обработки.
NB:
  • При сохранении протокола сохраняются всегда все имеющиеся записи независимо от состояния переключателя Отображать только ошибки.
  • Сбойные записи в протоколе всегда имеют в начале строки последовательность символов [!], в то время как успешно добавленные начинаются с символов [+]. Это позволяет легко искать, скажем, ошибки в протоколе (если только у вас нет обыкновения включать в имена файлов эти последовательности символов).
Контекстное меню (доступное по нажатию правой кнопки мыши в списке) предоставляет дополнительные опции:
Копировать Копирует текущую выбранную строкув буфер обмена.
Найти... Отображает стандартный диалог поиска текста, позволяющий произвести поиск в списке протокола, например, какого-либо имени файла.

Вы можете просмотреть протокол до завершения добавления всех файлов, прервав процесс добавления и нажав кнопку Далее. После этого можно (нажав Назад) вернуться к странице добавления и продолжить процесс. {[%hh_undosup_ru]} {[%hh_seealso_ru]} Мастер добавления изображений

Эта страница Мастера отображает процесс выполнения операций:

  • Эскиз изображения, с которым производится операция
  • Индикатор прогресса (состояния)
  • Счётчик успешно обработанных изображений
  • Счётчик сбоев при обработке
{[%hh_h2:Элементы интерфейса]}
Прервать/Продолжить Эта кнопка позволяет прервать выполнение текущей операции и, нажатием кнопки Далее >, перейти к протоколу обработки. Когда процесс прерван, повторное нажатие этой кнопки возобновляет его.

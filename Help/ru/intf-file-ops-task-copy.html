Мастер операций с файлами изображений :: Операция копирования файлов изображений

Эта операция позволяет получить копию набора файлов изображений в заданной папке, и поддерживает множество опций. В частности, она позволяет подготовить фотоальбом для записи на сменный носитель, такой как CD, DVD и др.

Чтобы выполнить эту операцию, необходимо запустить Мастер операций с файлами изображений и на странице выбора задачи выбрать Копировать файлы изображений в папку.

Далее последовательность страниц такова:

  • Выбор изображений для операции
  • Выбор папки назначения
  • Опции копирования/перемещения
  • Опции создания CD/DVD
  • Процесс выполнения операции
  • Протокол операции

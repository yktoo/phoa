Мастер операций с файлами изображений

Этот мастер служит для выполнения различных операций, затрагивающих файлы изображений. Вызывается он из режима обзора выбором команды меню Сервис | Операции с файлами... или с помощью комбинации клавиш Ctrl+R.

Мастер позволяет выполнять следующие задачи:

  • Копирование файлов изображений в заданную папку
  • Перемещение файлов изображений в заданную папку
  • Удаление ненужных изображений вместе с файлами
  • Перестройка эскизов изображений
  • Восстановление ссылок на файлы изображений

Описания страниц Мастера:

  1. Выбор задачи
  2. Выбор изображений для операции
  3. Выбор папки назначения
  4. Опции копирования/перемещения
  5. Опции создания CD/DVD
  6. Опции удаления
  7. Опции восстановления ссылок на файлы
  8. Процесс выполнения операции
  9. Протокол операции
